# CardOS2-Flexible-Micropython-OS
CardOS2 is a lightweight and modular operating system for handheld running Micropython. Cos2 was primarily made to run on the CBIT (another project of mine).

The goal of CardOS2 system is to be as flexible as possible. This is achieved through input/output drivers, a global settings system, and hardware abstraction.
