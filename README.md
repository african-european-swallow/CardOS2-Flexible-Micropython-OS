# CardOS2-Flexible-Micropython-OS
CardOS2 is a lightweight and modular operating system for handheld running Micropython. It was primarily made to run on the CBIT (another project of mine).

The goal of CardOS2 system is to be as flexible as possible. This is achieved through input/output drivers, a global settings system, and hardware abstraction.

CardOS2 is designed to be independent of the hardware. As long as the hardware has compatible drivers, the same operating system, and applications, they can run on different boards, displays, and input devices with little or no modification.

To get more information on how to use CardOS2 please go to the docs folder.
